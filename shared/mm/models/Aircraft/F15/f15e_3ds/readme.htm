F-15E Strike Eagle

Version 1.3

Very detailed USAF fighter-bomber with LANTIRN pods. Speedbrake, intake ramps, engine fans, landing gear, refuel door, canopy, seats and all control surfaces are separate objects for animating. Includes a separate set of landing gear doors in the closed position for fast in-flight setup.

Version 1.3 updates:

- Smoothed geometry throughout exterior fuselage and canopy
- Added missing canopy mirrors
- Made minor corrections to fuselage geometry
- Added afterburner flameholder detailing to engine exhaust tubes
- Added full intake trunking to the engine compressor faces
- Separated the intake ramps so they can be positioned as on the actual aircraft
- Modified all tires adding tire tread geometry

Version 1.2 updates:

  •  Re-created the tail-mounted RWR sensors
  • Updated all texture maps to better match the overall Gunship Gray paint scheme

Version 1.1 updates:

  •  Smoothed geometry throughout exterior fuselage
  • Corrected gray cockpit color within texture maps
  • Re-created all exterior texture maps at 200% original size
  • Made minor corrections to fuselage geometry
  • Fixed incorrect surface color inside speedbrake well
  • Simplified engine exhaust geometry and added bitmap to simulate more realistic
    details
  • Added separate model file which features closed landing gear doors for animators
    who want to simply delete the lowered gear to quickly convert the model to gear-up
    configuration
